# Prerequisites
- node 7.10.0
- yarn 0.24.5

## Building

    make

## Running

    ./bin/itemize <count>

## Removal

    make clean

## Notes
This was a LOT of trouble to setup and it's probably not idiomatic. I'm honestly surprised it works, but it does compile and bundle the app to a single .js file, with a .map, which is great.
